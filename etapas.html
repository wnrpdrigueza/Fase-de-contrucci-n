Siete etapas a tener en cuenta para desarrollar software
1. Análisis de requerimientos

En esta etapa se debe tener clara la visión de lo que se necesita es por esta razón que es muy importante analizar y reconocer lo que en verdad vamos a necesitar. En esta etapa es donde permite al ingeniero especificar las características operacionales del software, indica el software con otros elementos del sistema y así mismo se establece las restricciones que debe cumplir el software.		 2. Especificación

En esta etapa es donde se debe describir detalladamente el software a ser escrito, de una forma clara.
3. Diseño y arquitectura

Se determina de forma general de como funcionara el software, se indica claramente la estructura, funcionamiento e interacción entre las partes del software.		 4. Programación

Es la parte donde se trabaja el diseño y la arquitectura a código. Es la parte más obvia del trabajo de ingeniería de software y la primera en que se obtienen resultados notorios.		5. Prueba

Consiste en comprobar que el software responda y realice correctamente las tareas indicadas y esperadas en el momento del diseño, aplicando las pruebas de calidad de software necesarias para que con ellas se pueda ayudar a reducir los riesgos, logrando que se identifiquen los defectos antes de que se ejecuten.
6. Documentación y validación

una vez finalizadas y aprobadas las pruebas del software se valida y se empieza a documentar un manual técnico con la realización del manual de usuario, y posiblemente un manual técnico con el propósito de mantenimiento futuro y ampliaciones al sistema.		7. Mantenimiento

En esta etapa se realizan un mantenimiento correctivo con el fin de resolver errores, de la misma forma realizar un mantenimiento evolutivo que mejore la funcionalidad y/o dar respuesta a nuevos requisitos, en esta etapa de software se pueden involucrar cambios al software en orden de corregir defectos y dependencias encontradas durante su uso tanto como la adición de nueva funcionalidad para mejorar la usabilidad y aplicabilidad del software.
